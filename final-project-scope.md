### **Project Name:** Project 03 - Final Project
## **Project Purpose**
The purpose of this project is to scope out the goals, requirements and limitations for the initial build of The Coffee Nook website. 
## **Description**
The primary objective is to create a visually appealing, two-page static website for a fictional business that demonstrates proficiency in HTML structure, CSS styling and deployment via Github.
## **Desired Results**
- **Home Page**: Summary of the business and primary offerings.
- **About Page**: Business history and operational hours table.
- **Navigation**: A link to the about page from the home page.
- **Visual Design (CSS)**: 
    - Implementation of a "cozy" color palette (e.g., browns, creams, or forest greens).
    - Custom typography using web-safe fonts.
    - Basic layout spacing using Padding and Margins.
- **GitHub Deployment**: Hosted and viewable via GitHub Pages.
- Use **Semantic HTML** tags.
- Use an **External CSS Stylesheet** (`style.css`) linked to both HTML pages.
- Maintain a clean file structure.
## **Exclusions**
- **Contact Form**: Functional backend or email submission.
- **Online Ordering**: E-commerce functionality or shopping carts.
- **JavaScript**: Interactive elements like pop-ups.
